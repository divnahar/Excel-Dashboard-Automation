# Excel-Dashboard-Automation

This project showcases a macro-enabled Excel dashboard that uses PivotTables, PivotCharts, and slicers to generate interactive reports. It includes VBA automation that dynamically connects slicers to multiple PivotTables using a dashboard checkbox.

- Separate exported VBA module for code review
